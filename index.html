Vlixes e-commerce Sport Shop


Esta aplicación es un e-commerce de ropa deportiva. La idea es que los usuarios puedan comprar ropa deportiva y luego puedan ver el carrito de compras.
Tecnologías:
- Para la interfaz: TypeScript - React - Redux - CSS Modules - Sass
- Para el backend: Node.js - Express.js - Sequelize - Stripe
- En la base de datos: PostgreSQL
- El deployado: Heroku (back) y Vercel (front)
puedes jugar con la app Aqui!

Henry's Dogs


Aplicación en la cual se puedan ver distintas razas de perro junto con información relevante de las mismas utilizando la api externa [the dog api](https://thedogapi.com/) y a partir de ella poder, entre otras cosas:
- Buscar perros - Filtrarlos / Ordenarlos - Agregar nuevos perros
Principales tecnologías:
- React - Redux - Express - Sequelize - Postgres.
Puedes utilizar la app Aqui.
